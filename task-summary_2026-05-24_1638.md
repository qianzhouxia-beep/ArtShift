# PixelCode 完整代码结构分析（最终版）

## 项目概述
将开源项目 screenshot-to-code 改造为付费网站「PixelCode」，域名 pixelcode.api-tokenmaster.com。核心功能：截图/设计稿 → AI → 代码。AI模型采用Gemini。

---

## 一、整体架构

```
Browser (https://pixelcode.api-tokenmaster.com)
  │
  │  WebSocket: ws://pixelcodebackend.zeabur.app/generate-code
  │  JSON: FullGenerationSettings
  │
  ▼
Backend Pipeline (6层Middleware)
  │
  ├── WebSocketSetupMiddleware      ← 接受连接
  ├── ParameterExtractionMiddleware ← 提取+验证参数
  ├── StatusBroadcastMiddleware    ← 广播 variantCount
  ├── PromptCreationMiddleware     ← 构建 Prompt
  ├── CodeGenerationMiddleware     ← 模型选择 + Agent 运行
  └── PostProcessingMiddleware     ← 后处理
      │
      ▼
  Agent.run(model, prompt_messages)
      │
      ▼
  ProviderSession (Gemini/OpenAI/Anthropic)
      │
      ▼
  流式 Tool Calls:
    create_file → HTML 提取 → file_state
    edit_file   → 字符串替换
    generate_images (需 API Key)
    remove_background (需 Replicate Key)
```

## 四、Agent 执行引擎

### 4.1 工具调用循环 (agent/engine.py)
```python
_run_with_session():
  max_steps = 20
  for _ in range(max_steps):
    turn = await session.stream_turn(on_event)
    
    if not turn.tool_calls:
      return _finalize_response(turn.assistant_text)
    
    for tool_call in turn.tool_calls:
      tool_result = await tool_runtime.execute(tool_call)
      await _send("setCode", tool_result.updated_content)
    session.append_tool_results(turn, executed_tool_calls)
```

### 4.2 代码流式预览 (_stream_code_preview)
- 工具：`create_file` content 增量发送
- 策略：max 18 chunks，min 200 chars/step，0.01s 间隔
- 触发时机：工具开始时 + 增量 ≥ 40 chars

### 4.3 HTML 提取优先级 (codegen/utils.py)
1. `<file path="...">...</file>` 标签
2. ` ```html ... ``` ` Markdown 代码块
3. `<!DOCTYPE html>...</html>`
4. `<html>...</html>`
5. 原始文本（兜底）

## 六、429 错误根因分析

### 6.1 现象
- 上传截图后，WebSocket 返回 429 错误
- Gemini API 配额耗尽

### 6.2 根因
- Gemini 免费用户配额限制严格
- 每次截图 → base64 编码 → 大量 tokens
- 快速测试多次触发配额限制

### 6.3 解决方案
1. **获取付费 API Key**（推荐）：Google AI Studio 申请付费 API Key
2. **等待重置**：免费配额每日重置
3. **降低频率**：减少测试次数
4. **Key 轮换**：准备多个 API Key

### 6.4 关键环境变量
```
GEMINI_API_KEY=your_key  # 在 Zeabur Backend 环境变量中配置
```
