## 一、项目概述

### 1.1 产品定位

**一句话定位**：面向全球消费者 / 创意买家的 AI 驱动个性化定制商品电商平台。

用户无需设计技能，只需描述想法或上传照片，AI 即可生成专属图案，再一键定制到 T 恤、帽子、杯子、手机壳等实体商品，由 POD 平台（Printful）完成生产与全球配送。

### 1.2 核心价值主张

- **零门槛设计**：用自然语言描述想法，AI 即刻生成设计稿
- **所见即所得**：AI 生成图可实时预览在产品上的效果
- **一件起订，无库存**：POD 模式，无需囤货，按单生产
- **全球可达**：Printful 覆盖欧美 20+ 国家本地配送 3-7 天到货

## 二、用户旅程地图（从访问到收货）

```
[访问] → [注册/登录] → [AI 生成设计 或 上传照片处理] → [选产品]
→ [预览确认] → [下单支付] → [平台接单生产] → [物流配送] → [确认收货]
                                              ↓
                                        [评价晒图/再设计]
```

### 阶段 1：访问与引导
- 用户从社交媒体 / Google 广告 / SEO 落地页进入
- 首页即展示 AI 生成演示（示例设计 + 产品预览），无需注册即可体验 AI 生成预览
- 落地页 CTA：**"Create Yours →"**

### 阶段 2：设计创作
- **路径 A（文字→图）**：输入 prompt（如 "a cute cat wearing sunglasses on a beach"），AI 生成多张图案供选择
- **路径 B（照片→设计）**：上传照片，AI 执行风格迁移（宠物卡通化、个人画像艺术化等）
- 设计不满意可一键重新生成（消耗 AI 积分）
- 支持基础编辑（调整颜色、文字叠加）

### 阶段 3：选品与预览
- 选择产品类型（T 恤 / 卫衣 / 帽子 / 杯子 / 手机壳等）
- AI 图案自动贴合产品模板，实时显示"上身效果图"
- 选择颜色、尺寸、数量

### 阶段 4：下单与支付
- 加入购物车 → 结算页
- 支持 PayPal / Stripe 信用卡（支持全球主流支付）
- 显示预估配送时间（Printful 实时接口）

### 阶段 5：生产与物流
- 订单推送至 Printful API（自动化）
- Printful 本地印刷 + 包装
- 用户收到邮件追踪号，可随时查看物流状态

### 阶段 6：收货与复购
- 提醒用户评价、上传"上身照"
- 社交分享奖励（下次消费折扣码）
- 推荐相似设计或产品（复购引导）

## 三、POD 平台对比

### 横向对比

| 维度 | Printful | Printify | Gooten | Gelato |
|------|----------|----------|--------|--------|
| **质量** | ⭐⭐⭐⭐⭐ 行业最佳 | ⭐⭐⭐⭐ 良好 | ⭐⭐⭐⭐ 良好 | ⭐⭐⭐⭐ |
| **覆盖地区** | 北美/欧洲/澳洲本地仓 | 全球（部分中国发货） | 全球 | 全球 |
| **平均配送时间** | 3-7 天（本地仓） | 7-14 天（部分跨境） | 5-10 天 | 5-10 天 |
| **API 完善度** | ⭐⭐⭐⭐⭐ REST + webhook | ⭐⭐⭐⭐ REST | ⭐⭐⭐⭐ REST | ⭐⭐⭐⭐ |
| **与 Shopify/WooCommerce 集成** | 原生支持，开箱即用 | 原生支持 | 原生支持 | 原生支持 |
| **SKU 数量** | 300+ 产品 | 900+ 产品（更多低价选项） | 100+ | 100+ |
| **定价模式** | 按单收费（含生产+配送） | 按单收费（略低，但运费另计） | 按单收费 | 按单收费 |
| **白标体验** | 是（无 Printful 标识） | 是 | 是 | 是 |
| **退款/售后政策** | 严格（破损可免费重发） | 严格 | 较宽松 | 较宽松 |

### ⚠️ 关键风险点

- **Printify 部分产品从中国发货**：会导致配送时间过长，影响用户体验
- **Printful 质量更稳定**，但价格略高

### 🏆 推荐：Printful（主平台）

**理由**：
1. 质量最稳定，客诉率低，适合作为品牌初期建立口碑
2. API 文档完善，集成开发成本低
3. 全球本地仓（美/英/德/澳/加），配送快，用户体验好
4. 支持自动化 webhook，无需轮询

**备选**（第二期扩展）：
- 如果 SKU 数量不够，可接入 Printify 作为补充
- 长期可考虑 Gelato（欧洲本地化更强）

**Printful 接入要点**：
- 需要在 Printful 创建店铺并获取 API Token
- 通过 Printful API 创建订单（POST /orders）
- 监听 webhook 接收生产/物流状态更新
- 订单确认后 2-3 小时 Printful 开始生产

## 四、商业模式与定价策略

### 4.1 商业模式画布

```
收入来源 = 商品销售利润 + AI 增值服务（可选）

核心成本：
├── POD 生产成本（Printful 结算价）
├── 支付手续费（Stripe 2.9% + $0.30）
├── 平台抽成（若有）/ 服务器成本
└── 营销成本（按 CPA 计量）

利润 = 售价 - POD 成本 - 支付手续费 - 预估损耗（3%）
```

### 4.2 产品定价策略

#### 基础定价公式

```
售价 = POD 成本 × 2.5 ~ 3.0 倍（溢价系数）

示例（T 恤）：
├── Printful 结算价（空白衫 + 印刷）：~$12-15
├── 平台运营成本（支付手续费等）：~$1.5
├── 合理利润（毛利 40%）：~$8-10
└── 建议售价：$29.99 - $34.99
```

#### 产品定价矩阵（MVP 阶段）

| 产品类型 | POD 成本参考 | 建议售价 | 毛利率目标 |
|---------|------------|---------|----------|
| T 恤（棉质） | $12-15 | $29.99 | ~50% |
| 卫衣 | $18-22 | $44.99 | ~52% |
| 帽子（棒球帽） | $8-10 | $24.99 | ~55% |
| 陶瓷杯 | $7-9 | $22.99 | ~56% |
| 手机壳 | $6-8 | $19.99 | ~53% |
| 环保袋 | $5-7 | $19.99 | ~58% |

**定价原则**：数字心理学（$29.99 vs $30），不设奇数定价感强的 $31/$33。

### 4.3 AI 生成收费模式

#### 方案 A：免费基础 + 付费增值（推荐）

| 层级 | 描述 | 价格 |
|------|------|------|
| 免费用户 | 每天 3 次 AI 生成机会，下载低分辨率图（72dpi） | $0 |
| 基础付费（$9.99/月） | 每月 50 次生成，下载高清图（300dpi），无水印 | $9.99/月 |
| Pro 订阅（$19.99/月） | 无限次生成，优先队列，高级风格包，商业授权 | $19.99/月 |

#### 方案 B：纯按次付费（更透明）

| 动作 | 价格 |
|------|------|
| AI 生成 4 张图（任选 1 个满意） | $1.99 |
| 高清下载（300dpi，无水印） | +$1.99 |
| 风格升级包（Pro 风格） | +$0.99 |

**推荐方案 A（月订阅）**：
- 降低用户决策摩擦，预估月度留存率更高
- AI 生成成本可控（使用 Stable Diffusion 开源模型，服务器成本 $0.05-0.1/次）

### 4.4 收入预期（MVP 阶段）

假设：
- 月访问量 5,000，转化率 3%，即月订单 150 单
- 平均客单价 $32（基于产品组合）
- 月 GMV = $4,800
- 毛利率 50%，月利润约 $2,400（不含营销）

## 八、增长策略（海外获客）

### 8.1 冷启动阶段（第 0-3 个月）

#### 渠道 1：社交媒体内容营销（低成本高杠杆）

| 平台 | 策略 | 目标 |
|------|------|------|

| **Pinterest** | 上传高质量产品设计图（"Custom Cat T-shirt Design"） | SEO 长期流量 |

#### 渠道 2：付费广告（第 1 个月开始）

| 渠道 | 预算建议 | 目标 | KPI |
|------|---------|------|------|
| **Meta Ads（Facebook/Instagram）** | $500-1000/月 | 流量 + 转化 | CPA < $20，ROAS > 3 |
| **Google Ads（搜索）** | $300-500/月 | 高意向关键词 | CPA < $15 |
| **TikTok Ads** | $300/月（测试） | Z 世代用户 | CTR > 3% |

#### 渠道 3：SEO 长尾流量

- 目标关键词：`custom printed t-shirts online` / `AI design t-shirt maker`
- 每月发布 4 篇博客（SEO + 内容营销）

### 8.2 增长飞轮（第 3-6 个月）

```
优质产品 → 客户晒图（UGC） → 社媒传播 → 新用户
    ↓
用户复购 / 分享折扣码
    ↓
收入增长 → 加大广告投放
```

**关键增长杠杆**：
- **晒图返券**：用户上传产品照，返 $5 折扣码（刺激 UGC + 复购）
- **Referral 计划**：邀请 1 位好友，双方各得 $10 折扣码
- **Product Hunt 发布**：上线当天集中曝光
- **Reddit 营销**：在 r/tshirts、r/CustomTshirts、r/PrintOnDemand 社区分享

### 8.3 数据驱动优化

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 转化率（访→下单） | > 3% | 优化落地页 + AI 生成体验 |

## 九、技术架构概览（开发团队参考）

```
前端：Next.js（SSR + SEO） + Tailwind CSS + React
AI 生成：Stable Diffusion XL（自托管 GPU）或 DALL-E 3 API（OpenAI）
图像处理：Pillow / Sharp（模板贴合） + ControlNet（风格化）
后端：Node.js / Python FastAPI
数据库：PostgreSQL（Supabase 或自托管）
存储：Cloudflare R2 / AWS S3（设计图）
支付：Stripe（主）+ PayPal
POD 集成：Printful API + Webhook
邮件：Resend / SendGrid
部署：Vercel（前端）+ Railway / Render（后端）
```
