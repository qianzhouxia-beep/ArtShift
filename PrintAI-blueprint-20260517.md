## 一、项目定位（一句话）

**面向海外消费者的 AI 驱动个性化定制商品平台** — 用户描述想法或上传照片，AI 即刻生成专属图案，一键定制到 T 恤/帽子/杯子/手机壳，由 Printful 完成生产与全球配送。

## 三、技术方案（全栈架构师结论）

### 技术栈
| 层级 | 技术选型 | 理由 |
|------|---------|------|
| 前端 | Next.js 14 + TypeScript + Tailwind + shadcn/ui | SSR 利于 SEO，全栈减少运维 |
| 状态管理 | Zustand | 轻量，无 Redux 复杂性 |
| 后端 | Next.js API Routes + 独立 Express（AI 推理） | 统一语言，降低维护成本 |
| 数据库 | PostgreSQL + Redis | 关系数据 + 缓存/队列 |
| ORM | Prisma | 类型安全，迁移方便 |
| AI 生成 | **Replicate API（SDXL）** | $0.002-0.005/张，无需自管 GPU |
| POD | Printful API | 完善文档 + Webhook 自动同步 |
| 支付 | Stripe Checkout | 国际支付主流，Webhook 可靠 |
| 图片存储 | Cloudinary 或 AWS S3 | CDN 加速 |
| 邮件 | Resend 或 SendGrid | 事务性邮件 |
| 部署 | Linux + Nginx + PM2 + Let's Encrypt | 基于已有服务器 |
| CI/CD | GitHub Actions | 自动部署 |

### AI 方案对比（结论：Replicate SDXL）
| 方案 | 成本/张 | 质量 | 推荐度 |
|------|---------|------|--------|
| DALL-E 3 API | $0.04 | 极高 | ⭐⭐⭐ 预算充足才用 |
| **Replicate SDXL** | **$0.002-0.005** | **高** | **⭐⭐⭐⭐⭐ 推荐** |
| Midjourney API（第三方） | 高 | 极高 | ⭐⭐ 不稳定 |
| 自部署 SDXL | 低（固定） | 高 | ⭐⭐⭐⭐ 后期降本用 |

### 系统架构（文字版）
```
用户浏览器（Next.js 前端）
    ↓ API 请求 / Webhook 回调
Next.js API Routes（BFF 层）
    ↓
后端服务层：AI Service / Order Service / Payment Service
    ↓
PostgreSQL（用户/订单/设计）+ Redis（队列/缓存）
    ↓
外部服务：Replicate API / Printful API / Stripe API / Cloudinary
```

### 数据库核心表（Prisma）
- `User`：用户基础信息
- `Design`：AI 生成的设计稿（关联用户、prompt、图片 URL、放置参数）
- `Order`：订单（关联用户、设计、Printful 订单 ID、状态、物流号）
- `Product`：产品模板（关联 Printful 产品 ID、预览模板）
- `Variant`：产品规格（颜色、尺寸、价格）

### 开发优先级（3 个 Phase）

#### Phase 1：MVP（4-6 周）✅ 核心流程跑通
1. 基础前端（Next.js + Tailwind）
2. AI 文生图（Replicate API，4 张图 < 30 秒）
3. 风格选择器（8-10 种预设风格）
4. 产品选型（5 种产品：T 恤/卫衣/帽子/杯子/手机壳）
5. 产品预览（Canvas 图像合成，实时效果图）
6. Stripe 支付集成（测试模式）
7. Printful 订单同步（沙盒环境）
8. 用户注册/登录（邮箱 + Google OAuth）
9. 基础订单状态追踪（Printful Webhook）
10. 部署到生产环境（Nginx + PM2）

#### Phase 2：功能完善（3-4 周）
1. AI 照片风格化（上传照片 → ControlNet 风格迁移）
2. 设计编辑器（拖拽调整位置/大小）
3. 设计收藏（个人画廊）
4. 订单管理页面（用户查看状态）
5. 邮件通知（下单确认、生产进度、物流到达）
6. 折扣码系统
7. 产品颜色/尺寸选择（Printful 库存 API）

#### Phase 3：优化与扩展（持续）
1. 更多产品类型
2. AI 智能改图（自然语言指令修图）
3. 社交分享功能
4. AR 预览（Three.js + WebXR）
5. 自部署 SDXL（降本）
6. 移动端 App（React Native）

## 四、MVP 功能清单（优先级 P0）

| 功能 | 描述 | 技术实现 |
|------|------|---------|
| AI 文生图 | 输入文字，生成 4 张图案 | Replicate SDXL API |
| 风格选择器 | 8-10 种预设风格 | LoRA 微调或风格标签注入 |
| 产品选型 | 5 种产品 | Printful 产品库 API |
| 产品预览 | 实时效果图 | Canvas API 图像合成 |
| 购物车 + 结算 | 标准电商流程 | Stripe Checkout |
| 订单推送 Printful | 自动同步 | Printful REST API |
| 物流追踪 | 显示配送状态 | Printful Webhook |
| 用户登录 | 邮箱 + Google | NextAuth.js |

**MVP 明确不做**：AI 视频、AR 试穿、社交社区、B2B 批量、移动 App、多语言、虚拟货币
