## 📚 [LRN] 学习点提炼

### 学习 1：SVG Sprite Sheets 的正常行为

**知识点**：
- SVG Sprite Sheets 直接打开时**总是显示为空白**
- 这是**正常行为**，不是错误

**原因**：
- 它们只包含 `<symbol>` 定义
- 需要通过 `<use href="icons.svg#icon-name" />` 嵌入到其他 SVG 中

**应用**：
- 以后遇到"空白 SVG"时，先检查是否是 Sprite Sheet
- 不要直接认定为错误，检查文件内容是否有 `<symbol>` 定义

---

### 学习 2：Chat UI 附件预览的局限性

**知识点**：
- 聊天系统的附件预览功能**无法访问本地文件系统路径**

**现象**：
- 当我引用本地文件（如 `C:\...\favicon.svg`）时
- 聊天 UI 会显示"文件已删除"

**本质**：
- 这是**聊天 UI 的限制**，不是文件真的被删除

**应用**：
- 以后遇到类似"文件已删除"提示时
- 先用 `Test-Path` 或 `ls` 验证文件实际存在性
- 不要轻信 UI 的预览状态

---

### 学习 3：浏览器 Favicon 优先级

**知识点**：
- 浏览器**优先加载 SVG favicon**，而不是 PNG

**影响**：
- 即使 `index.html` 同时引用了 PNG 和 SVG
- 浏览器可能只显示 SVG

**应用**：
- 如果必须使用 PNG favicon（比如用户提供了官方 PNG）
- 应该**移除 SVG 引用**以避免冲突
- 最佳实践：只引用一个 favicon 格式
